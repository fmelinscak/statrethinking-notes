n_grid <- 50
data1 <- list(W = 3, L = 0)
data2 <- list(W = 3, L = 1)
data3 <- list(W = 5, L = 2)

p_grid <- seq(0, 1, length.out = n_grid)

prior <- rep(1, n_grid)

posterior1_unstd <- dbinom(data1$W, data1$W + data1$L, p_grid) * prior
posterior1 <- posterior1_unstd / sum(posterior1_unstd)

2H1 Given information: \[ P(A) = P(B) = 0.5 \\ P(T|A) = 0.1 \\ P(S|A) = 1 - P(T|A) = 0.9 \\ P(T|B) = 0.2 \\ P(S|B) = 1 - P(T|B) = 0.8 \]

Information required: \[ P(T_2|T_1) = ? \]

We can extend the conversation (opposite of marginalization) to include the intervening variable of species: \[ P(T_2|T_1) = P(T_2|T_1,A)P(A|T_1) + P(T_2|T_1, B)P(B|T_1) \] We will additionally assume that the births are conditionally independent (i.e., when we know the species, the fact that the first birth was a twin, gives us no additional information): \[ \begin{aligned} P(T_2|T_1) &= P(T_2|A)P(A|T_1) + P(T_2|B)P(B|T_1) \\ &= 0.1 \times P(A|T_1) + 0.2 \times P(B|T_1) \\ \end{aligned} \]

Now using the Bayes theorem:
\[ \begin{aligned} P(A|T_1) &= \frac{P(T_1|A)P(A)}{P(T_1|A)P(A) + P(T_1|B)P(B)} = 1/3 \\ P(B|T_1) &= 1 - P(A|T_1) = 2/3 \end{aligned} \] Finally, we obtain: \[ \begin{aligned} P(T_2|T_1) &= P(T_2|A)P(A|T_1) + P(T_2|B)P(B|T_1) \\ &= 0.1 \times 1/3 + 0.2 \times 2/3 \\ &= 1/6 \end{aligned} \]

2H2. Information required: \[ P(A|T_1) = ? \] We can use Bayes theorem as in H1: \[ P(A|T_1) = \frac{P(T_1|A)P(A)}{P(T_1|A)P(A) + P(T_1|B)P(B)} = 1/3 \]
